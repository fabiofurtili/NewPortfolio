# Portfólio — Fábio Furtili

Site/portfólio pessoal com apresentação profissional, projetos e seção de contato. Inclui easter eggs com Snake e player de áudio.

## Estrutura
- `index.html` — página principal
- `assets/css/style.css` — estilos
- `assets/js/main.js` — scripts gerais
- `assets/js/easter-eggs/` — códigos dos easter eggs
- `assets/img/` — imagens dos projetos
- `assets/sound/` — áudios dos easter eggs

## Funcionalidades
- Layout responsivo
- Projetos com preview de imagem e lightbox
- Contatos com ícones (Font Awesome)
- Easter eggs:
  - Snake (sequência: `↑ ↓ ↑ ↑ ↓`)
  - Player “Minha História” (5 cliques no nome)

## Como usar
Basta abrir o `index.html` no navegador.

## Observações
- O player “Minha História” usa o Green Audio Player via CDN.
- Ícones do Font Awesome via CDN.
